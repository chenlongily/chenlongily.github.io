再见孙悟空

发表于 2018-06-19 | 分类于 随笔

研究生毕业了，写一篇文章记录下，毕竟很多事情，不记录下就会慢慢遗忘。

开始的开始

一切都要从2014年的年的8月份开始说起，彼时的我正值大四上半学期的暑假，面临着毕业，找了一份实习，每天一大早坐一个多小时的公交去某公司实习，下午又坐着公交回来，傍晚时分，坐着公交穿过北京的每一条街道，望着熙熙攘攘、灯红酒绿的北京，听着汪峰的《北京北京》，北漂的感觉瞬间涌上心头。

9月份，开学之前，因为有个专业实习（公费旅游）要去大连，向公司请了假。在愉快的的实习（公费旅游）结束之后，就开学了，然而北化变态就是在大四下学期还有一堆课程要修，于是乎开学之后，我就不能保证每周四天的实习时间了，在挣扎了一段时间之后，我就辞职了，算是结束了人生中的第一份工作。

这个时候已经要10月份了，但我还没有要考研的打算，我认为应该早点去社会工作更好，但是周围有不少同学在准备考研，加上父母也希望我读个研，我就开始纠结了。和大部分的人可能不同的是，我知道只要我静下心来，不考985的研究生而考本校的研究生的话，肯定能考上，即使还有100天不到。因为我通常只需要花20%的时间就能把一件事做到60%，但我缺少耐心和勤奋把一件事做到极致（低调地装一波逼）。所以我要说服的是自己，为什么要考研？上研究生的意义是什么？我说服了自己很久，最终也没有很好的理由说服自己。其实即使到要毕业的今天，我可能明白了些研究生的意义，但我也不能说当时考研就是比工作更好。回过头来说，最终让我决定要考研的一个可能原因是我那迷信的老妈找人给我算过命，说我22岁要上研究生。我一想既然已经命中注定就服从命运安排吧。

写在26岁

发表于 2018-05-07 | 分类于 随笔

贰零壹捌年阴历三月二十，人生第25个年头已经走完，感谢陪我过生日的小伙伴们和你们的礼物，记录一下此时的心境，等老了的时候能有点东西用来下酒。

个人认为20-30岁是人生最重要的十年，这十年你需要慢慢摆脱对父母和别人的依赖，作为一个成年人来面对这个社会和世界；这十年你需要完成你的学业，学习一身的本领来面对以后的各种挫折和磨难；这十年你需要寻找一个伴侣来和你一起走完剩下的路；这十年你需要开始确立你的事业并慢慢承担一个家庭的责任。

这十年我已经走完了一半，刚刚要完成学业。在这样的一个年龄，好像前途充满希望，一切皆有可能，又对未来充满迷茫，感觉最终还是要归于平凡。在这样的一个年龄，有的同龄人已经结婚生娃，孩子都开始打酱油了，有的同龄人自己创业已经实现财富自由。在这样一个年龄，能力配不上野心又不肯努力，宝剑尚未配妥，出门便已是江湖。在这样一个年龄，青春慢慢从身边流走，理想似有似无，不停徘徊在理想与现实之中。

你好，我叫陈龙

发表于 2018-01-20 | 分类于 随笔

我的个人博客终于被我建好了，终于我在互联网上有了一个自己的地盘来发出自己的声音。欢迎参观，电脑网页版体验更佳。

https://chenlongily.github.io/

现在我的心情就如当年毛爷爷在开国大典上向全世界宣布：

中华人民共和国成立了！

然而成功的花人们只惊羡她出现时的明艳，当初她的芽儿，浸透了奋斗的泪泉， 洒遍了牺牲的血雨。这个博客的建立背后，有一段血泪史，待我慢慢道来。

那是一个秋天，风儿那么缠绵……咳咳，说人话。好吧那是8月的某天，刷着网页的我突然看到一个大神的博客，于是乎想我是不是也能自己做一个。于是就发了个朋友圈，立了个flag说要做一个自己的博客，当时的我宛如戏台上的老将军身上插满了flag，也不在乎多插一个。

恩，说干就干，知道大概的方向是用github能做一个静态的博客，不用自己买云服务器和域名。然后就开始就是开始学习建网站需要的编程语言了。不出意外，前两天满腔热血，第三天就完全忘了这回事。嗯，间歇性踌躇满志，持续性混吃等死。就这样大概过了两周，我听了个腾讯课堂的公开课，觉得凭我的意志力自学是不可能的，然后就花了6000块（别问我哪来这么多钱，我的肾有点痛……）报了一个线上前端的培训班。对于一直想学编程的我，这点钱……还是很多的，尤其是想到可以换一个手机或者笔记本电脑的时候~~。

我常用的几个神器

发表于 2018-01-19 | 分类于 随笔

火车抢票神器-12306bypass

http://www.12306bypass.com/

春节将近，大家都需要抢票回家，试过各种抢票软件，这个应该是最给力的，帮我和同学都抢到票了，最牛逼的一次在于抢到两张卧铺票。软件是免费的，不过唯一不足的是免费需要手动输入验证码，自动识别验证码需要付费。不过只要刷到票的第一时间输入验证码，基本都能抢到票。

百度云下载神器-度盘下载器

http://linesoft.top/bbs/forum.php?mod=viewthread&tid=64

百度云盘是百度不多的良心产品了，丰富的资源，超大免费容量，但是百度云坑的是不是会员的话，下载速度出奇的慢。而度盘下载器能让你的下载速度轻松突破5M，不过有钱的还是尽量支持下百度会员。

网盘、学术搜索神器-虫部落

http://magnet.chongbuluo.com/

百度云盘很好，但是各种分享的资源哪里找，这个网站集成了各种网盘搜索网站，一般要看的电影、电视剧什么的都能搜到。这个网站还有学术搜索、快搜和数据搜索等相似的集成。自己慢慢探索吧。

Markdown 编辑神器-Typora

https://www.typora.io/

markdown 可能不是程序员的人，基本很少了解，大家可以先去百度一下，了解一下markdown是什么以及它的语法。也可以在马克飞象上试试自己用markdown写一篇文章，不出意外的话，你会爱上他的，慢慢放弃word和text这种编辑器了。

翻墙神器-ishadow

https://my.ishadowx.net/

如果不知道什么是翻墙？自己先百度一下。对于翻墙要求不是很高的人来说蓝灯和ishadow都是不错的选择。蓝灯最容易，ishadow需要下载软件，再扫描这个网站免费的ss二维码，而且每6个小时服务器会重启和更换密码，需要重新扫描一下二维码。不过你懂的，这个网站有时候会挂。
