我常用的几个神器

发表于 2018-01-19 | 分类于 随笔 |

火车抢票神器-12306bypass

http://www.12306bypass.com/

春节将近，大家都需要抢票回家，试过各种抢票软件，这个应该是最给力的，帮我和同学都抢到票了，最牛逼的一次在于抢到两张卧铺票。软件是免费的，不过唯一不足的是免费需要手动输入验证码，自动识别验证码需要付费。不过只要刷到票的第一时间输入验证码，基本都能抢到票。

百度云下载神器-度盘下载器

http://linesoft.top/bbs/forum.php?mod=viewthread&tid=64

百度云盘是百度不多的良心产品了，丰富的资源，超大免费容量，但是百度云坑的是不是会员的话，下载速度出奇的慢。而度盘下载器能让你的下载速度轻松突破5M，不过有钱的还是尽量支持下百度会员。

网盘、学术搜索神器-虫部落

http://magnet.chongbuluo.com/

百度云盘很好，但是各种分享的资源哪里找，这个网站集成了各种网盘搜索网站，一般要看的电影、电视剧什么的都能搜到。这个网站还有学术搜索、快搜和数据搜索等相似的集成。自己慢慢探索吧。

Markdown 编辑神器-Typora

https://www.typora.io/

markdown 可能不是程序员的人，基本很少了解，大家可以先去百度一下，了解一下markdown是什么以及它的语法。也可以在马克飞象上试试自己用markdown写一篇文章，不出意外的话，你会爱上他的，慢慢放弃word和text这种编辑器了。

翻墙神器-ishadow

https://my.ishadowx.net/

如果不知道什么是翻墙？自己先百度一下。对于翻墙要求不是很高的人来说蓝灯和ishadow都是不错的选择。蓝灯最容易，ishadow需要下载软件，再扫描这个网站免费的ss二维码，而且每6个小时服务器会重启和更换密码，需要重新扫描一下二维码。不过你懂的，这个网站有时候会挂。

pdf 在线神器 - smallpdf

https://smallpdf.com/

平时是要pdf转word 图片，拆分合并pdf什么的。很多pdf 的软件要不功能不全，要不需要付费。有了这个网站，妈妈再也不担心pdf的各种操作了。

快捷键练习神器 - shortcutfoo

https://www.shortcutfoo.com/

几乎我们常用的一些软件都有快捷键，快捷键可以极大的提高我们的工作效率，一个软件的高手基本都是靠快捷键来进行各种操作，这个网站提供了我们常用软件快捷键的练习。还能设置定时发送邮件提醒你，进行复习。不过对于懒的人，最好不要设置，不然它没发一次邮件就会提醒一次你是一个懒人这个事实，别问我怎么知道的，我已经关闭了所有提醒。

设计神器

http://hao.uisdc.com/

https://www.behance.net/

https://www.pinterest.com/volusion/

http://www.zcool.com.cn/

http://huaban.com/

作为一名略懂设计的工科文艺青年，我也是曾经迷恋过设计的，ps ppt 的设计都学过，以上网站是学设计的人经常要逛的网站吧，收集素材，吸取灵感，你值得拥有。

暂时先安利这些了，基本是我这两年收藏的各种神器，希望对大家有一定帮助。
